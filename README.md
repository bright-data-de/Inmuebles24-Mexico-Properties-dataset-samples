Dieses **Inmuebles24 Mexico Properties Dataset-Beispiel** umfasst **1.236 Datensätze** und wurde mit der **Bright Data API** extrahiert.

## Datenpunkte

| Feldname | Beschreibung |
|------------|-------------|
| `url` | Die eindeutige Webadresse für das Immobilieninserat auf Inmuebles24. |
| `Title` | Der Haupttitel des Immobilieninserats, wie vom Verkäufer angegeben. |
| `GeneratedTitle` | Ein systemgenerierter Titel, der Immobiliendetails wie Typ und Größe zusammenfasst. |
| `Description` | Eine detaillierte Beschreibung der Immobilie einschließlich Merkmale und Standort. |
| `Seller` | Der Name oder die Kennung der Person oder Organisation, die die Immobilie verkauft. |
| `Precio` | Der angegebene Preis der Immobilie in der angegebenen Währung. |
| `Publicado_hace` | Die seit der Veröffentlichung des Immobilieninserats verstrichene Zeit, ausgedrückt in Tagen. |
| `CreatedDate` | Das Datum, an dem das Immobilieninserat auf der Plattform erstellt wurde. |
| `Visualizaciones` | Die Anzahl der Aufrufe des Immobilieninserats. |
| `Dimension_terreno` | Die gesamte Grundstücksfläche der Immobilie in Quadratmetern. |
| `Dimension_propiedad` | Die bebaute oder überdachte Fläche der Immobilie in Quadratmetern. |
| `Estacionamiento` | Die Anzahl der zur Immobilie gehörenden Parkplätze. |
| `Banos` | Die Anzahl der Badezimmer in der Immobilie. |
| `Dormitorios` | Die Anzahl der Schlafzimmer in der Immobilie. |
| `Antiguedad` | Das Alter der Immobilie, typischerweise in Jahren. |
| `pictures` | Links oder Kennungen für Bilder der Immobilie, die im Inserat enthalten sind. |
| `videos` | Links oder Kennungen für Videos der Immobilie, die im Inserat enthalten sind. |
| `Tipo_de_inmueble` | Die Art der Immobilie, z. B. Wohnung, Haus oder Gewerbefläche. |
| `Tipo_de_operacion` | Die Art der Transaktion, z. B. Verkauf oder Miete. |
| `Tipovendedor` | Die Art des Verkäufers, z. B. Eigentümer oder Immobilienmakler. |
| `Phone1` | Die primäre Kontakttelefonnummer für Anfragen zur Immobilie. |
| `whatsApp` | Die WhatsApp-Kontaktnummer für Anfragen per Sofortnachricht. |
| `Operacion` | Gibt die Art des Vorgangs an, z. B. Kauf oder Leasing. |
| `Region` | Die übergeordnete geografische Region, in der sich die Immobilie befindet. |
| `Zona` | Die spezifische Zone oder Nachbarschaft der Immobilie. |
| `Ciudad` | Die Stadt, in der sich die Immobilie befindet. |
| `Provincia` | Die Provinz oder das Bundesland, in dem sich die Immobilie befindet. |
| `Direccion` | Die vollständige Adresse der Immobilie. |
| `Superdestacado` | Gibt an, ob das Inserat auf der Plattform hervorgehoben ist. |
| `postingId` | Eine eindeutige Kennung für das Immobilieninserat. |
| `postingCode` | Ein Code, der das Immobilieninserat repräsentiert und häufig intern verwendet wird. |
| `publisher` | Der Publisher oder die Plattform, über die das Inserat beworben wird. |
| `premier` | Gibt an, ob das Inserat einen Premier-Status oder eine erhöhte Sichtbarkeit hat. |
| `URL` | Die direkte URL zum Immobilieninserat. |
| `aviso_finalizado` | Gibt an, ob das Inserat abgeschlossen oder geschlossen wurde. |
| `currency` | Die Währung, in der der Preis der Immobilie angegeben ist. |
| `latitude` | Der geografische Breitengrad der Immobilie. |
| `longitude` | Der geografische Längengrad der Immobilie. |


## Anwendungsfälle

### 1. Analyse des Immobilienmarkts
Analysieren Sie Immobilientrends und Preise in Mexiko, um Immobilienfachleuten dabei zu helfen, datengestützte Investitionsentscheidungen zu treffen.

### 2. Wettbewerbsvergleich
Vergleichen Sie Inserate in verschiedenen Regionen, um die Wettbewerbspositionierung zu bewerten und Immobilienmarketingstrategien zu optimieren.

### 3. Planung der Stadtentwicklung
Nutzen Sie geografische und immobilienspezifische Daten, um Stadtplaner bei der Gestaltung nachhaltiger und effizienter städtischer Infrastrukturen zu unterstützen.
